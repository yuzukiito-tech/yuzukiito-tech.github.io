開発環境

eclipse:Java SE 17
実行時:Java 17 JDK必須

起動したときに初めて見るページ。
アプリケーションの簡単な説明が書いてある。
名前入力ページに遷移。
全体を通して、メイン画面をバツ閉じすると、プログラムが終了する。

主人公の名前を入力するページ。
テキストボックスに何も入力されていないと遷移しないようになっている。
デフォルト名なし。
名前入力決定(同じページ)と前のページに遷移。

名前が入力されていると見えるページ。
ページ自体は↑のものと同じ。
入力された主人公の名前を含めたストーリーラインが読める。
ストーリーラインが読めるようになったら、ゲームを始められる。
↑と同じく、名前入力決定と前のページ遷移ができる。

ゲームが始まると見えるページ。マップページ。
ゲーム本編。
マップが見える。
マップ移動のためのボタンと話しかける・調べるなどができるcheckボタン、
ステータス確認などが行えるメニューフレームを開くmenuボタンがある。
調べた結果が左下の欄に現れ、テキストボックスでやり取りができる。
町・ダンジョンはマップの外に出ることで、世界地図に、世界地図から町・ダンジョンの位置に重なることで、それぞれのマップに遷移する。
町以外の世界地図、ダンジョンの地図の時に魔物と遭遇し、バトルページに自動で遷移する。

マップ表示説明

共通
O：自機位置
|:壁。通り抜け不可。
_:通れる箇所

町マップ
n:宿屋。主人公の体力・魔力を最大値まで回復する。調べることで自動的にお金を払って回復される。
s:道具屋。金額と引き換えに道具を買うことができる。

ワールドマップ
t:町。
d:ダンジョン。
自機と重なることでマップ遷移する。
～:海。通り抜け不可。

ダンジョンマップ
T:宝箱。アイテムが入っている。たまに入っていないことがある。
N:開けた宝箱。
B:ダンジョンボス。倒すことでゲームクリアになる。

マップページから開くことのできるメニューページ。
メインのマップページとは別のフレーム。
主人公の現在の体力・魔力・レベル・所持金が開いたときに見える。
メニューから所持している道具や特技が確認・使用できる。
また、詳細なステータスも確認可能。
閉じるときはバツ閉じ。

どうぐ

主人公がもっている道具の一覧を表示する。範囲の都合上５つまでしか表示がされない。
１つを選択し、使う・使った時の効果の確認・捨てることができる。
使う際は主人公から主人公に対して使うことになる。

とくぎ

主人公が持っている特技の一覧を表示する。
１つを選択し、使う・使った時の効果の確認ができる。
戦闘用特技など、メニュー画面から使えない特技もある。

ステータス

名前をはじめとし、最大値と現在の体力・魔力や攻撃力などが数値としてみることができる。

メニューから道具を1段階開いた状態。

マップページから遷移したバトルページ
攻撃する・特技を使う・道具を使う・逃げることができる。
テキストボックスで数字をもらってテキストが変化する。
戦闘中の結果も出す。
戦闘後の結果やレベルアップした結果なども、表示する。
戦闘が終了したら、マップページに遷移する。
終了時、主人公が負けていたら最終ページに遷移する。

攻撃する

戦闘している中から1人を選んで攻撃する。
攻撃力＋いくらかの補正をかけている。

特技を使う

主人公が使える特技の中から１つを選んで使う。
特技によって内容が異なる。
初期は何も使えないが、レベルが上がるごとに使えるものが増える。

道具を使う

主人公がもっている道具の中から1つを選んでつかう。
特技によって内容が異なる。
初期は何も持っていないが、魔物との戦闘や購入で増やすことができる

にげる

戦闘している中から逃走する。
先頭の敵一人のすばやさと比較し、成功判定を行う。

戦闘で主人公の体力が0になった時か、ボス戦に勝った時にバトルページから遷移する、最終ページ
遷移したときの状態によって、表示するメッセージを変える。
スタートページに遷移できる。
遷移するときにstaticなパラメータを初期化する。
